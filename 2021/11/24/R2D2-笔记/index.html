R2D2-笔记

发表于 2021-11-24 更新于 2021-11-25 阅读次数： Valine：

网上看了一圈R2D2的讲解，也没有具体的实现，原论文也没有过多的介绍实现细节。看了一下 https://github.com/neka-nat/distributed_rl/ 对R2D2的实现，总结成笔记。
https://xlnwel.github.io/blog/reinforcement%20learning/R2D2/
https://xlnwel.github.io/blog/reinforcement%20learning/Apex-DQfQ/

首先，总体框架同Apex一样，多个actor，一个learner，搭配parameter server。
不同点首先一个是网络，Apex实现的时候使用的是DQN,或dueling DQN， R2D2使用了 Dueling LSTM DQN，在CNN与全连接之间添加了一层LSTM cell（不是LSTM layer）
dueling的实现，主要是一个优势函数：

buffer type length what’s inside Detail
Step buffer deque n_step(3) Transiton(S,A,R,D) Once Full add the n-step return of first element to seq buffer
recurrent_state_buffer list n_step(3) hidden state Once Full add the n-step return of first element to seq buffer
Sequence buffer list n_burn_in + n_sequence Transiton(S,A,R,D) Once Full push all into local_memory,clear to overlap size
Local memory ReplayMemory size(1000) Sequence(sequence_buffer,recurrent_state_buffer[0]) if size>_batch_size,sample batch_size push to PrioritizedMemory then clear memory
PrioritizedMemory PrioritizedMemory 50000 Sequence(sequence_buffer,recurrent_state) prioritized_sample batch to learner
