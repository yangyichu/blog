贝叶斯滤波，目的是构造后验估计的概率密度函数。

对许多问题来说，每次新的观测后都需要一个新的估计，因此一个简单的解决方法是使用递归滤波器(recursive filter)：A recursive filtering approach means that received data can be processed sequentially rather than as a batch so that it is not necessary to store the complete data set nor to reprocess existing data if a new measurement becomes available

非线性跟踪问题，最优估计在特定的约束条件下是可以直接求解的，例如卡尔曼滤波Kalman filter和网格滤波grid-based filter。但很多情况下不可解，此时就需要扩展卡尔曼滤波，近似网格滤波和粒子滤波。
